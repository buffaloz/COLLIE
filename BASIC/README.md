## BASIC 설명

```
1. editor.py : sd카드의 시리얼넘버를 추출하여 perm.c의 변수에 직접 기입.

2. perm.c : 추출된 시리얼넘버와 저장된 시리얼넘버 변수를 비교하여 특정변수 리턴.

3. permm.py : perm.c의 리턴값 패스 및 리턴값이 없으면 에러생성하여 시스템 강제종료.

4. permm.pyc : permm.py의 바이너리 파일로 permm.py 내용을 숨김.

5. setting.sh : 보안스크립트 적용과정 자동실행 및 파일자동삭제.

6. setup.py : perm.c를 파이썬 모듈화 시키는 셋업파일.
```
