## FINAL 설명

```
1. Etc : 각종 유틸리티 스크립트 + 비 실행파일 모음 폴더.

2. based_files : 음성인식의 기본적인 설정 파일 + 데이터 베이스 폴더.

3. htk : HTK툴 모음 및 트레이닝 관련 스크립트 폴더.

4. workspace2 : 트레이닝된 음성모델을 포함하는 작업폴더. (추가설명 없음)

5. Realtime.py : 실시간 음성인식 + 음성 트레이닝 스크립트.

6. Realtime_BLE.py : 스마트폰 어플리케이션으로 LED 및 음성인식 제어가 가능.

7. apa102.pyc : 리스피커의 RGB_LED를 제어하는 파이썬 모듈.

8. hcopy.so : MFCC 파이썬 모듈.

9. hvite.so : 음성모델과 Tied-list를 이용하여 인식결과를 출력하는 파이썬 모듈.

10. install.py : 실행 환경설정 설치 스크립트.

11. install.sh : 실행 환경설정 설치 스크립트.

12. like.py : 음성인식 결과값의 확률분포를 출력하는 스크립트.

13. pixels.py : 리스피커의 RGB_LED를 직접 제어하는 스크립트.

14. player.py : 실시간 음성피드백 스크립트.

15. starter.py : 블루투스 페어링 + starter.sh 스크립트.

16. starter.sh : 음성인식기 + 음성피드백 실행 스크립트.
```
