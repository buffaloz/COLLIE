## htk 설명

```
1. HTKTools : HTK툴 모음 폴더. (추가설명 없음)

2. MakeDB.py : mfcc파일을 previousDB폴더에 복사하는 스크립트.

3. Rec_al.py : Trainning.py에서 녹음용 모듈.

4. Rec_tra.py : Trainning_sw.py에서 녹음용 모듈.

5. Trainning.py : 콘솔창에서 트레이닝이 가능한 스크립트.

6. Trainning_sw.py : LED와 버튼으로 제어하는 트레이닝 스크립트.

7. create.py : 트레이닝 과정에서 필요한 파일생성 및 경로관리 스크립트.

8. new_codetr.py : DB폴더의 mfcc파일 사용을 위한 경로관리 .

```
