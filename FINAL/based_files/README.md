## based_files 설명

```
1. previousDB : 기존 트레이닝에 사용된 mfcc파일 모음.

2. config.config : mfcc파일 생성을 위한 설정값 파일.

3. config1.config : hvite를 위한 설정값 파일.

4. config2.config : hvite를 위한 설정값 파일.

5. diction.txt : 발음사전 파일.

6. filescript.txt : wav파일을 mfcc화 하기위한 경로.

7. filescript1.txt : train.scp 경로.

8. gram : 명령어 총 갯수.

9. maketrihed : triphone 생성 스크립트.

10. mkclscript.prl : triphone 확률분포.

11. mkphones0.led : monophone으로 변환하는 스크립트.

12. mktri.led : triphone으로 변환하는 스크립트.

13. proto.txt : 음성학습모델용 파일.

14. tree.hed : tied-list 확률분포.

```
